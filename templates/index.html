About This System

Overview

This AI/ML system identifies exoplanets from space mission data using advanced machine learning techniques.

Supported Missions

  • Kepler: Primary mission (2009-2013) discovering thousands of exoplanets
  • K2: Extended Kepler mission (2014-2018)
  • TESS: Transiting Exoplanet Survey Satellite (2018-present)

Detection Method

The transit method detects exoplanets by measuring the decrease in stellar brightness when a planet passes between the star and the observer.

Key Features

  • Multiple ML algorithms (Random Forest, XGBoost, LightGBM, Neural Networks)
  • Automatic feature selection and preprocessing
  • SMOTE for handling class imbalance
  • Real-time predictions with confidence scores
  • Interactive visualizations
  • Hyperparameter tuning interface
